# Portfolio Prototype

This is a dynamic, data-driven portfolio website built with **Python**, **Flask**, and **Bootstrap 5**.
**This is not my actual portifolio, it is just a prototype**.

## Features
* **Dynamic Data Rendering**: Experiences and interests are managed via JSON files.
* **Mobile Responsive**: Fully responsive design using Bootstrap 5.
* **Dynamic Routing**: A single template handles multiple page redirections using URL parameters.
* **Interactive UI**: Includes flip-cards and a "copy-to-clipboard" email button.

## Project Structure
* **Backend**: Python / Flask
* **Frontend**: Jinja2 Templates, Bootstrap 5, CSS3
* **Data Storage**: JSON

## Deployment
* Deployed using Render. Can be accessed in https://projetopessoal-portifolioonline.onrender.com/.
